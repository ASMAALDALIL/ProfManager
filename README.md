# 🎓 ProfManager

**ProfManager** est une application Full-Stack innovante destinée à simplifier la gestion pédagogique des enseignants. Ce projet fait partie de mon parcours académique (ENSA) et intègre des technologies modernes pour répondre aux besoins du secteur éducatif.

## 🚀 Fonctionnalités
* **Gestion des étudiants** : Suivi des absences, comportement et participations.
* **Intelligence Artificielle** : Prédiction des résultats scolaires et aide à la décision.
* **Bilans Automatisés** : Génération de rapports semestriels.
* **Interface Bilingue** : Support complet Français et Arabe.

## 🛠️ Stack Technique
* **Frontend** : React.js, Tailwind CSS (PWA).
* **Backend** : FastAPI (Python).
* **Base de données** : PostgreSQL (Neon.tech).
* **Hébergement** : Firebase (Front) & Render (Back).

## 📂 Structure du projet
- `/backend` : API FastAPI et modèles SQLAlchemy.
- `/frontend` : Application React et gestion des langues.
